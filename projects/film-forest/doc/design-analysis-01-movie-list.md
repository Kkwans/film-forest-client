# 设计图分析 - 电影列表页(PC端)

> 分析日期: 2026-05-04
> 模型: mimo-v2.5
> 图片: design-movie-list-pc.png

## 一、顶部导航栏

- 左侧: 树形图标 + "影视森林" 文字 Logo
- 中部: 导航链接 -- 首页, 电影, 电视剧, 综艺, 动漫, 短剧
- 右侧: 搜索框(placeholder: 搜索影片、演员、导演) + 深色/浅色模式切换开关(月亮/太阳图标)
- 导航栏背景: 白色, 底部有细线分隔

## 二、筛选/分类栏

### 类型标签(横向排列)

- 全部(绿色背景, 选中态), 剧情, 科幻, 喜剧, 动作, 爱情, 悬疑, 恐怖
- 标签样式: 圆角标签, 未选中为灰色边框

### 筛选下拉(右侧)

- 年份, 地区, 语言 -- 下拉选择器
- 排序 -- 下拉菜单, 包含: 最新更新, 评分最高, 热度最高, 上映时间

## 三、电影卡片网格

### 布局

- PC端: 4列网格
- 卡片间距: 适中(约16-20px)

### 卡片结构(从上到下)

1. 海报图片
   - 左上角标签: "【电影】" (红色/橙色背景)
   - 右下角标签: "4K/蓝光" (蓝色/紫色小标签)
   - 海报比例: 约 2:3 (竖版海报)
2. 标题行
   - 左侧: 电影标题(黑色粗体)
   - 右侧: 评分数值(绿色, 如 6.0, 8.0, 4.5)
3. 元信息行
   - 格式: 年份 + 地区(如 "2023 · 中国")
   - 灰色小字
4. 资源链接行
   - 两个按钮: "U 磁力链" + "O 网盘下载" (带图标)
   - 按钮样式: 浅灰色背景, 圆角

### Hover 效果

- 卡片上移 8px
- 阴影加深
- 封面图片轻微放大

## 四、底部

- 版权信息: "影视森林 2026 免责声明: 本站仅聚合影视资源信息, 不提供存储、播放服务"
- 居中显示, 灰色小字

## 五、配色方案

- 背景: 白色(#ffffff)
- 卡片背景: 白色
- 主文字: 黑色/深灰
- 次要文字: 灰色(#666666)
- 强调色: 绿色(选中标签、评分)
- 标签色: 红色/橙色(电影标签), 蓝色(4K标签)
- 边框/分隔: 浅灰色(#e5e5e5)

## 六、响应式要点

- PC端: 4列网格, 导航栏水平排列
- 平板: 可能 3列
- 手机: 2列, 导航栏需要折叠/汉堡菜单
- 筛选栏: 手机端需要横向滚动或折叠

## 七、可复用的设计模式

1. 卡片组件: 海报 + 标签 + 标题 + 评分 + 元信息 + 资源链接
2. 筛选栏: 类型标签 + 下拉筛选器
3. 导航栏: Logo + 导航链接 + 搜索 + 模式切换
4. Hover 动效: 上移 + 阴影 + 缩放
