# 管理端全面 UX 优化 - SDD

> 创建: 2026-05-13
> 状态: 进行中

## 一、需求
管理端存在大量移动端适配和交互问题，需要全面优化：
- 表单弹窗在手机上挤成一团、重叠、超框
- Select 下拉在 Modal 内被裁剪
- 爬虫配置无法创建（Modal + Select z-index 冲突）
- 任务日志条数显示 0（未展开时未加载）
- 多处 grid 布局在小屏崩溃
- Modal/Select 组件使用硬编码深色颜色，浅色模式异常

## 二、问题清单
1. Modal 组件: 硬编码 zinc-900/zinc-700 颜色，不跟随主题
2. Modal 组件: overflow-hidden 裁剪内部 Select 下拉
3. Select 组件: z-50 在 Modal(z-9997) 内被裁剪
4. Select 组件: 硬编码 zinc 颜色
5. 表单布局: grid-cols-2 在 <375px 手机上太窄
6. 爬虫表单: Modal footer + 表单底部双重保存按钮
7. 爬虫日志: 展开前日志条数显示 0
8. 仪表盘: 内容分布 grid-cols-5 手机崩溃
9. 统计页: 概览 grid 和占比条在手机上异常
10. 资源页: 小屏优化

## 三、任务拆分
T1: 修复 Modal 组件（主题色 + z-index 策略 + 移动端布局）
T2: 修复 Select 组件（主题色 + Modal 内兼容）
T3: 修复爬虫页表单布局 + 去重保存按钮 + 日志计数
T4: 修复内容管理页表单布局
T5: 修复仪表盘响应式布局
T6: 修复统计页响应式布局
T7: 修复资源管理页小屏优化
T8: 修复设置页小屏优化
T9: 构建验证 + 部署
