## 二、技术规格（Spec）

### 技术栈
- 前端: Next.js 16 + TypeScript + Tailwind CSS 4
- 后端: SpringBoot 3.2 + MyBatis-Plus 3.5
- 数据库: MySQL 8 (film_forest)
- 部署: Docker on NAS (192.168.5.110)

### 参照模块（Harness）
- 用户端 UI 设计风格（深色主题、卡片布局、动画效果）
- admin-ui/src/app/crawler/page.tsx（表格+弹窗模式）

## 三、任务拆分（Tasks）

### T1: 仪表盘排版样式重构 + 爬虫状态列表乱码修复
- **参照**: 用户端首页设计
- **内容**:
  1. 仪表盘页面重新设计（统计卡片 + 最近内容 + 爬虫状态 + 快捷操作）
  2. 修复爬虫状态列表中文乱码（双重编码问题）
  3. 参考用户端 UI 风格（深色主题、卡片布局、动画效果）
- **验证**: 中文正常显示，布局美观，移动端适配
- **状态**: ⏳

- **验证**: 弹窗样式美观，字段完整，移动端正常显示
