# 记忆规范

## 三类记忆

| 类型 | 更新频率 | 存储位置 |
|------|----------|----------|
| 当日对话记忆 | 每对话有新增信息即更新 | memory/YYYY-MM-DD.md |
| 长期记忆 | 永久有效、主人要求牢牢记住的 | memory/MEMORY.md |
| 规则记忆 | 规范性准则 | rules/*.md |

## 记忆更新原则
1. **新信息 → 当日记忆**：每对话有新的信息点，立即更新当日记忆
2. **永久信息 → 长期记忆**：跨对话长期有用的 → 写入MEMORY.md
3. **主人明确要求 → 长期记忆**：主人说"记住" → 立刻写长期记忆
4. **规范规则 → rules/**：执行准则放rules/，MEMORY.md中引用

## 当日记忆格式
```markdown
# 2026-04-22 对话记忆

## 新增信息
- [信息点1]
- [信息点2]

## 待跟进
- [ ] 待办1

## 结论
- [重要结论]
```

## 长期记忆原则
- 精简必要信息，不大段引用
- 包含索引（MEMORY.md），具体内容在对应文件
- 主人明确说过的重要事项必须永久记住
